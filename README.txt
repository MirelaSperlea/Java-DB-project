   Online auction application for mobile phones. 
    This project is made in the NetBeans IDE program and the JAVA programming language, the database is made in SQLiteStudio.
    The app has a database of users who can both bid on a mobile phone and post a new auction online. 
    Users can bid on multiple mobile phones that belong to a single category characterized by the phone brand.
    The following data is retained for each mobile phone: model, year of manufacture, etc. 
    Also, each mobile phone is associated with an auction that includes the following details: auction duration, seller, auction price, etc.
    For completed auctions, there is a record of them in which the following are stored: buyer, date, final price, etc.
    In the Project document there is a presentation of the application and the database.
