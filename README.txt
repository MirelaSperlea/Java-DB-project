    Aplicatie de licitatii online pentru telefoane mobile. 
    
    Acest proiect este realizat in programul NetBeans IDE si limbajul de programare JAVA, baza de date este realizata in SQLiteStudio.
    Aplicația are o bază de date a utilizatorilor care pot atât să liciteze pentru un telefon mobil cât să publice o noua licitație online. 
    Utilizatorii pot licita la mai multe telefoane mobile care aparțin unei singure categorii caracterizată de marca telefonului.
    Pentru fiecare telefon mobil se rețin următoarele date: model, an fabricație etc. 
    De asemenea fiecare telefon mobil este afiliat unei licitații ce cuprinde următoarele detalii: durata licitații, vânzătorul, prețul licitație etc.
    Pentru licitațiile finalizate există o evidență a acestora în care se stochează: cumpărătorul, data, preț final etc.
